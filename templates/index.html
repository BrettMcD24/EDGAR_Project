How It Works

From ticker to analysis
in under 5 seconds

01

Enter a ticker

Type any U.S. exchange-listed stock symbol. We resolve it to the SEC's internal CIK identifier instantly.

→
02

We fetch from EDGAR

Our server pulls XBRL-tagged financial data directly from SEC EDGAR's free API — no scraping, no intermediaries.

→
03

Clean & compute

Raw XBRL tags are mapped to human-readable labels, scaled to USD millions, and ratios are computed automatically.

→
04

Analyse & export

Browse the tables, switch between statements, and download any view as a clean CSV for Excel or Python.
